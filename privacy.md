# LiveWyr Privacy Policy

**Effective Date:** June 29, 2026
**Last Updated:** June 29, 2026

This Privacy Policy explains how LiveWyr collects, uses, stores, shares, and protects information when you use LiveWyr products, bots, apps, websites, tools, and related services, including LiveWyr, LiveWyr Legacy, LiveWyr GameDay Ops, LiveWyr Media Room, LiveWyr Proof Engine, and any future LiveWyr-branded tools or services.

By using LiveWyr, you agree to the practices described in this Privacy Policy.

## 1. What LiveWyr Is

LiveWyr provides tools for online football gaming communities, including College Football online dynasty leagues. LiveWyr may help leagues manage records, coach profiles, score submissions, approvals, league history, media-style content, screenshot verification, game-week activity, and other league workflows.

LiveWyr operates primarily through Discord and may process information from Discord servers where the bot is installed.

LiveWyr may collect and store the following types of information.

### Discord and Server Information

* Discord server IDs
* Discord channel IDs
* Discord role IDs
* Discord user IDs
* Usernames, display names, nicknames, and avatars when needed for display
* Server names and basic server settings
* Role and permission settings related to LiveWyr

### League Configuration Data

* League name
* League type
* Current season
* Current week
* Commissioner role
* Assistant role
* Output channels
* Enabled features or plan settings
* Setup status
* Feature preferences

### Coach and League History Data

* Coach/user profiles
* Team assignments
* Team history
* Game results
* Scores
* Opponents
* Season and week history
* Head-to-head records
* Awards
* Championships
* Rivalries
* Leaderboards
* Notes submitted by commissioners or authorized users

### Score Submissions and Approval Data

* Submitted scores
* Pending results
* Approved results
* Rejected results
* Edits and corrections
* Submitter information
* Approver information
* Timestamps
* Audit logs
* Rejection or edit notes, if provided

### User-Submitted Content

Depending on the features used, LiveWyr may collect:

* Text entered into commands
* Notes
* Game summaries
* Media prompts
* Press conference responses
* Screenshots
* Image attachments
* Proof submissions
* Links provided by users
* Feedback or support requests

### Usage and Technical Data

LiveWyr may collect limited technical and usage data, such as:

* Commands used
* Feature usage counts
* Error logs
* Bot performance logs
* Server activity related to LiveWyr features
* Timestamps of interactions
* Subscription or entitlement status
* Plan access information

### Payment and Subscription Data

If LiveWyr offers paid features, subscriptions, add-ons, or purchases, payment processing may be handled by Discord, Stripe, or another payment provider.

LiveWyr may store:

* Plan name
* Subscription status
* Purchase status
* Entitlement IDs
* Renewal or cancellation status

LiveWyr does not intentionally store full payment card numbers or banking information.

## 3. How We Use Information

LiveWyr uses collected information to:

* Provide bot features and league tools
* Set up and manage league profiles
* Track coach records and league history
* Process score submissions and approvals
* Maintain audit logs
* Display profiles, leaderboards, H2H records, awards, and championships
* Support future LiveWyr features that depend on shared league data
* Generate league content when requested
* Verify screenshots or submitted proof when those features are enabled
* Manage subscriptions, plans, and feature access
* Prevent abuse or misuse
* Troubleshoot bugs and technical issues
* Respond to support requests
* Improve LiveWyr features and reliability

## 4. How LiveWyr Uses Discord Data

LiveWyr uses Discord data only as needed to provide its features.

For example:

* Server IDs identify which league a record belongs to
* User IDs connect coaches to records even if usernames change
* Role IDs determine who can approve scores or manage settings
* Channel IDs determine where bot messages should be posted
* Message or attachment data may be used when a user submits proof, screenshots, receipts, or other content

LiveWyr does not sell Discord user data.

## 5. User IDs and Display Names

LiveWyr stores Discord user IDs because usernames, nicknames, and display names may change.

User IDs help preserve accurate records over time, including coach profiles, game history, awards, and head-to-head records.

Display names may be shown in bot messages, profiles, leaderboards, and league history outputs.

## 6. Screenshot and Attachment Data

Some LiveWyr features may allow users to submit screenshots, images, or other attachments for score verification, stat tracking, proof, receipts, or league content.

When submitted, LiveWyr may store:

* The uploaded file or a reference to it
* The submitter
* The related game, season, week, user, or team
* Extracted text or interpreted data
* Approval status
* Audit history

Screenshot and attachment features may require commissioner or assistant approval before data becomes official.

Do not submit screenshots or files containing sensitive personal information.

## 7. AI and Automated Processing

Some LiveWyr features may use AI, OCR, image analysis, or automated processing to generate summaries, interpret screenshots, create posts, or assist with league content.

Information submitted to those features may be processed by third-party AI or infrastructure providers as needed to generate the requested output.

AI and OCR outputs may be inaccurate. Users and commissioners should review important results before accepting them as official.

## 8. When We Share Information

LiveWyr may share information in the following situations:

### Within the Discord Server

League data may be displayed to members of the Discord server where LiveWyr is installed, depending on command use and server settings. Examples include profiles, leaderboards, approved scores, awards, and league history.

### With Service Providers

We may use third-party providers for hosting, databases, storage, AI processing, analytics, payments, logging, or support.

These providers may process information only as needed to provide their services.

### For Legal or Safety Reasons

We may disclose information if required by law, legal process, platform policy, security investigation, abuse prevention, or to protect the rights, safety, or integrity of LiveWyr, users, or others.

### Business Transfers

If LiveWyr is transferred, sold, merged, or reorganized, information may be transferred as part of that transaction.

## 9. What We Do Not Do

LiveWyr does not:

* Sell personal data to advertisers
* Use league data to target unrelated advertising
* Intentionally collect full payment card numbers
* Intentionally collect sensitive personal information
* Require users to provide real names unless they choose to do so in Discord
* Access private Discord servers unless the bot is installed there
* Read all server messages by default unless a feature specifically requires message access and the bot has permission

## 10. Data Retention

LiveWyr keeps information as long as needed to provide services, maintain league history, comply with legal obligations, resolve disputes, prevent abuse, or operate the bot.

League history data may be retained for as long as the league continues using LiveWyr or until deletion is requested by an authorized server owner or commissioner.

Some logs may be retained for a shorter period for troubleshooting and security.

## 11. Data Deletion

A server owner or authorized commissioner may request deletion of LiveWyr data associated with their server.

Deletion requests may include:

* Removing league setup data
* Deleting coach profiles
* Deleting game history
* Deleting screenshots or proof records
* Removing audit logs where appropriate

Some data may be retained if needed for legal, security, fraud prevention, billing, abuse prevention, or backup reasons.

To request deletion, contact LiveWyr Support using the contact information below.

## 12. User-Level Requests

If you are a Discord user and want information associated with your Discord user ID removed or corrected, contact your league commissioner or LiveWyr Support.

Because LiveWyr records are often part of shared league history, removing one user’s data may affect league records, game history, H2H results, awards, or audit logs. We may need to coordinate with the server owner or commissioner before removing shared league records.

## 13. Data Security

We use reasonable technical and organizational measures to protect information collected by LiveWyr.

However, no system is perfectly secure. We cannot guarantee absolute security of data stored or transmitted through Discord, LiveWyr, third-party providers, or the internet.
